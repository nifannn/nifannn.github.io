问题

在上一场编程竞赛中你帮了Julia很大的忙，这次她也想让你帮忙！

参赛者的总分是他所有比赛获得的最大分的和。写一条查询语句打印hacker_id， name，和总分，根据总分降序排序。如果总分相同，根据hacker_id升序排序。排除总分为0的参赛者。

输入格式

下面的表包含了竞赛相关数据：

  • Hackers: hacker_id是参赛者id，name是参赛者姓名。

  • Submissions: submission_id是提交记录的id，hacker_id是提交者id，challenge_id是赛题id，score是该次提交的得分。

说明

参赛者4071 在赛题 19797 和 49593 中提交了答案，总分 = 95 + max(43,96) = 191。

参赛者74842 在赛题 19797 和 63132 中提交了答案，总分 = max(98,5) + 76 = 174。

参赛者84072 在赛题 49593 和 63132 中提交了答案，总分 = 100 + 0 = 100。

参赛者 4806， 26071， 80305，和49438的总分可以用类似的方法计算。
