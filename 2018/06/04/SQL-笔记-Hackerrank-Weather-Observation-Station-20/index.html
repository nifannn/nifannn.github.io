问题

中位数（median）定义为分割一个数据集较大的一半和较小的一半的一个数。查询STATION表中北纬（Northern Latitudes (LAT_N)）的中位数，并保留4位小数。

注意： 本题不允许使用Oracle的方法。

输入格式
STATION 表描述如下：

STATION
Field Type
ID NUMBER
CITY VARCHAR2(21)
STATE VARCHAR2(2)
LAT_N NUMBER
LONG_W NUMBER

LAT_N：北纬 LONG_W：西经

对于奇数个元素的数组，中位数是第ceil(n/2)个数。对于偶数个元素的数组，中位数是第n/2个数和第n/2 + 1个数的平均数。例如，一个数组有11个数，中位数是排序后的第6个数。对于一个有12个数的数组，中位数是第6和第7个数的平均数。但是如果从0开始计数呢？一个11个数的数组，索引变为0，1，2，…, 10，同时中位数的索引为5，正好是10的一半。另外，对于一个12个数的数组，索引为0，1，2，…，11，中位数是索引为5和6的数的平均数，5和6分别是floor(11／2),ceil(11／2)。所以，可以先将数组排序，然后用一个变量记录索引（从0开始），然后找出索引为ceil(i/2)和floor(i/2)的数（i为最后一个数的索引大小），再计算平均值作为中位数。
