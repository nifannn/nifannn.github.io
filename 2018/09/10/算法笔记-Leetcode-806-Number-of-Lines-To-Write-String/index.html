问题描述

将给定字符串S中的字母从左到右分行输出。每一行最大宽度100单位，如果输出某一个字母导致宽度超出100单位，则写在下一行中。本题给出一个数组widths，其中widths[0]是’a’的宽度，widths[1]是’b’的宽度， …，widths[25]是’z’的宽度。

现在回答两个问题：会有多少行包含至少一个S的字母，并且最后一行的宽度为多少单位？将答案作为一个长度为2的整数列表返回。

例子 :
输入:
widths = [10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10]
S = "abcdefghijklmnopqrstuvwxyz"
输出: [3, 60]
说明:
所有字母的长度均为10。要写出所有26个字母，需要写满两行，再加60单位的一行。

例子 :
输入:
widths = [4,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10,10]
S = "bbbcccdddaaa"
输出: [2, 4]
说明:
除了’a’之外的字母长度均为10，
“bbbcccdddaa” 会占据9 * 10 + 2 * 4 = 98单位。
最后一个’a’写在第二行因为第一行只剩2个单位。
所以答案是2行，再加第二行的4个单位。

注意:

  • S的长度区间为[1, 1000]。
  • S只包含小写字母。
  • widths是长度26的数组。
  • widths[i]的范围是[2, 10]。

解法

基本思路

需要用到两个变量，一个记录行数，一个记录这一行的宽度。先初始化，行数为1，宽度为0。因为先从第一行开始，并且当前没有放入任何字母。然后遍历字符串的每一个字母，找出对应宽度，尝试放入当前行中。如果宽度超过100，就将字母放入下一行，意味着行数加1，宽度设为该字母的宽度。

Python 实现

时间复杂度

O(N)。N是字符串长度，因为要遍历字符串的每一个字母。

空间复杂度

O(1)。使用一个长度为2的列表进行重要数值的记录。
