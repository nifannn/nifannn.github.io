问题

给定一个表格，BST，包含两列： N 和 P，N 表示二叉树一个节点的值，P 是 N 的父节点。

Column Type
N Integer
P Integer

写一条查询语句，得出二叉树中节点类型并按节点值的大小排序。按如下方式输出结果:

  • Root: 如果节点是根节点。
  • Leaf: 如果节点是叶子节点。
  • Inner: 如果节点既不是根节点也不是叶子节点。

输入样例

N P
1 2
3 2
6 8
9 8
2 5
8 5
5 null

输出样例

1 Leaf
2 Inner
3 Leaf
5 Root
6 Leaf
8 Inner
9 Leaf

分析

可以使用 IF() 函数解决这个问题。首先，需要返回N和节点的类型，然后根据N的值排序，可以把查询语句写成"SELECT N, IF() FROM BST ORDER BY N"。
然后，检查是否为根节点。如果P是null，则为根节点。查询语句可以写成"SELECT N, IF(P IS NULL, ‘Root’, IF()) FROM BST ORDER BY N"。
最后，检查是否为叶子节点。如果N出现在P列中，即一个节点是某个节点的父节点，则该节点为内部节点，否则为叶子节点。查询语句可写成"SELECT N, IF(P IS NULL, ‘Root’, IF(B.N IN (SELECT P FROM BST), ‘Inner’, ‘Leaf’)) FROM BST AS B ORDER BY N"。
或者写成 “SELECT N, IF(P IS NULL, ‘Root’, IF((SELECT COUNT(*) FROM BST WHERE P=B.N))) FROM BST AS B ORDER BY N”。
