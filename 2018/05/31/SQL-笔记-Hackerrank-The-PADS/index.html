问题

生成以下两个结果:

  1. 以字母表顺序查询 OCCUPATIONS 中的所有名字,并且后面跟着职业的首字母（用括号包围）。例如：AnActorName(A), ADoctorName(D), AProfessorName(P), and ASingerName(S).
  2. 查询 OCCUPATIONS 中每个职业的出现次数，升序排序，并按以下格式输出：

There are a total of [occupation_count] [occupation]s.

[occupation_count] 表示 OCCUPATIONS 中职业出现次数，[occupation] 为小写的职业名称。如果超过一个职业的出现次数相同，则按字母表顺序排序。
注意: 每类职业至少有两条记录。

输入格式
OCCUPATIONS 表格描述如下：

Column Type
Name String
Occupation String

职业为以下几种: Doctor, Professor, Singer 或者 Actor.

输入样例
一个 OCCUPATIONS 表格包含如下记录:

Name Occupation
Samantha Doctor
Julia Actor
Maria Actor
Meera Singer
Ashely Professor
Ketty Professor
Christeen Professor
Jane Actor
Jenny Doctor
Priya Singer

输出样例

Ashely(P)
Christeen(P)
Jane(A)
Jenny(D)
Julia(A)
Ketty(P)
Maria(A)
Meera(S)
Priya(S)
Samantha(D)
There are a total of 2 doctors.
There are a total of 2 singers.
There are a total of 3 actors.
There are a total of 3 professors.

解释
第一条查询语句的结果符合题目描述的要求。
第二条查询语句的结果先根据职业出现次数排序(2 ≤ 2 ≤ 3 ≤ 3),再根据字母表顺序排序(doctor ≤ singer, actor ≤ professor)。

分析

可以用CONCAT()函数解决这个问题。语法为’CONCAT(str1, str2, …)’，返回拼接参数后所得的字符串。
然后可以使用SUBSTR() 或者 LEFT() 函数来得到职业的首字母。SUBSTR()函数语法’SUBSTR(str, pos, len)’，返回一个长度为len，来自于字符串str的子字符串，开始位置为pos。LEFT()函数语法为’LEFT(str, len)’，返回从字符串str最左边起长度为len的子字符串，如果有参数为空则返回NULL。
另外,

  • OCCUPATIONS 表格 ==> FROM OCCUPATIONS
  • 名字按字母表顺序排序 ==> ORDER BY Name
  • 职业名字出现次数 ==> COUNT(Occupation) … GROUP BY Occupation
  • 小写的职业名称 ==> LOWER(Occupation)
  • 按出现次数升序排序 ==> ORDER BY COUNT(Occupation)
  • 如果出现次数相同，再按字母表顺序排序 ==>
    ORDER BY COUNT(Occupation), Occupation
